📋 手动安装步骤

步骤1: 以管理员身份打开PowerShell或命令提示符
步骤2: 导航到应用程序目录：
cd "C:\Program Files\Employee Monitor"
步骤3: 安装原生模块依赖：
cd native-event-monitor-win && npm install
步骤4: 编译原生模块：
npm run build
步骤5: 重启Employee Monitor应用程序
